# 6-Month Roadmap (Bachelor-equivalent → PhD prep)

## Phase 0: Reality Check (read once, don't re-read)

**6 months CAN deliver:**
- Full coverage of HKU BPsych core + MSocSc(Psy) compulsory theory
- Research methods + stats competency (your edge)
- 1 small empirical project (survey → analysis → report)
- Application-ready knowledge + research statement

**6 months CANNOT deliver:**
- Official transcript
- Lab experience
- Supervised practicum
- HKPS membership
- Clinical practice rights

**Honest path:** self-study → MSocSc(Psy) conversion → MPhil/PhD or DPsych.

---

## The 6 Months at a Glance

| Month | Theme | Core Topics | Key Deliverable |
|---|---|---|---|
| **1** | Foundations + Stats | Intro Psych, Research Methods, JASP/R | 1 simple survey + basic analysis report |
| **2** | Bio + Cognitive | Biopsychology, Cognitive Psych, Neuro | 1 concept map + 3–5 classic papers read |
| **3** | Dev + Social + Personality | Lifespan, Social, Big Five | 1500-word lit review on a subtopic |
| **4** | Abnormal + Electives | Psychopathology + chosen subfield | Case conceptualization (written) |
| **5** | Advanced + Academic Skills | Primary lit, Zotero, 3-pass, ethics | Capstone design finalized |
| **6** | Capstone + Application | Run project + PS + CV + research proposal | 4000–8000 word capstone + application package |

---

## Monthly Milestones (high-level)

### Month 1 — Foundations + Research Methods
- [ ] OpenStax Psychology 2e: finish chapters 1–5 (Intro) + 2 (Research Methods)
- [ ] JASP installed; complete "Learning Statistics with JASP" chapters 1–6
- [ ] 1 mini analysis on public dataset (Kaggle/OSF mental health)
- [ ] **Milestone:** Google Form survey designed (5–10 items, validated construct)

### Month 2 — Biological & Cognitive
- [ ] Noba Biological Psych module + Pinel selected chapters
- [ ] Noba Cognitive Psych module + Goldstein ch 1–6
- [ ] Draw concept map: "Brain → Behavior" (use Excalidraw or paper)
- [ ] **Milestone:** Read 3–5 classic papers (Müller-Lyer, Stroop, Kahneman & Tversky 1979, etc.)

### Month 3 — Developmental, Social, Personality
- [ ] Noba Developmental + Social + Personality modules
- [ ] Pick subtopic for 1500-word lit review (e.g., Big Five & decision making)
- [ ] **Milestone:** 1500-word lit review submitted (to self or peer)

### Month 4 — Abnormal + Subfield Elective
- [ ] Noba Psychopathology module
- [ ] Deep dive into chosen subfield (2–3 papers/week)
- [ ] **Milestone:** 1-page case conceptualization (DSM-5-TR style, informal)

### Month 5 — Advanced + Academic Skills
- [ ] Zotero setup with HKU Psy style (American Psychological Association 7th)
- [ ] 3-pass reading method internalized (5 papers minimum)
- [ ] CITI Program: complete "Responsible Conduct of Research" (free)
- [ ] **Milestone:** Capstone proposal finalized (1-page)

### Month 6 — Capstone + Application
- [ ] Run capstone study (survey live ≥2 weeks; N ≥ 50)
- [ ] Analyze with JASP/R
- [ ] Write 4000–8000 word report (capstone thesis style)
- [ ] Personal Statement (1000 words) + CV + research proposal (1 page)
- [ ] **Milestone:** Submit HKU MSocSc(Psy) application

## Recommended Weekly Time Split

| Block | Hours | Focus |
|---|---|---|
| **Mon–Fri evenings** | 2h × 5 = 10h | Reading + flashcard (Anki) + light practice |
| **Sat morning** | 3h | Stats / data analysis deep work |
| **Sun morning** | 3h | Paper reading + writing |
| **Sun evening** | 1h | Weekly review + plan next week |
| **Buffer** | ~3h | Catch-up, or push into capstone |

Total: ~20h/week. Adjust based on work intensity.

---

## What Success Looks Like

- **By Month 3:** you can read a psychology paper and understand the method, stats, and conclusion
- **By Month 6:** you can design a study, run it, analyze it, write it up
- **By application time:** you have a coherent narrative: engineer → self-taught → research-track psychologist
- **Bonus signal:** capstone topic + research proposal aligned with HKU faculty interests = stronger application
